Code Stack

This website was built using HTML, CSS, and JavaScript. A newsletter system is included that allows users to sign up and receive a welcome email and future updates using the MailerLite email service. Email addresses entered by users are sent to MailerLite, which manages subscriptions and sends automated messages. Styles are separated into organized CSS files so that layout, colors, and components are handled consistently across all pages. The navigation bar and footer are written once and loaded onto each page using JavaScript functions, instead of being copied directly into every HTML file, which helps to keep the site consistent across all pages.

Additional Libraries & Services Used

  • Google Fonts: Bebas Neue, Bree Serif
  • Inline SVG Icons: used for navigation and interface elements
  • MailerLite: newsletter signup and automated welcome emails
  • Vercel: hosting and deployment platform
